Variable

VBA中的变量，即可以根据算数值赋予不同值的量。用Dim 来表示

Variable用处

  1. 可以简化公式表达，使代码可读性提高
  2. 方便修改数值，避免重复修改
未使用变量X Sub 求相关值()

Cells(12, 2) = Cells(12, 1)*Cells(12, 1)*6

Cells(12, 3) =Cells(12, 1)*Cells(12, 1)*Cells(12, 1)

End Sub
使用变量X Sub 求相关值()

Dim x, y, z

x = Cells(12, 1)

y = x * x * 6

z = x * x * x

Cells(12, 2) = y

Cells(12, 3) = z

End Sub

Variable注意点

  • 使用变量前,需声明唯一变量（Option Explicit）
  • 变量名称不能用中文，可用英文或字母
